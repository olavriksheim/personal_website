Report

A review of this website.

User Interface

The header and menu doesn't take up too much space. The website is consistent. The website looks good. The navigation is clearly shown. The titles are descriptive. The website is responsive. It's adapted to all screen sizes. There's enough space betwen elemts, so it's easy to read.

GAS-analysis

Goal: the goal of the website is to tell about myself, and have a webvsite for the course.

Audiance: The audience of the website is the teacher, and people who want to know something about me.

Scope: The scope of the websiteis inofrmation about myself, my schedule, my projects and the different parts of the course.

Color Palette

The colors I have used are #333333, the primary color, and #EECE1A, the secondary color. In addition to this I have used the SASS functions: lighten() and darken(), to get lighter and darker colors. For the text I have used white. I have used these colors because I think they look good and are easy to look at.

Font

The font I have used is "Segoe UI". This is a sans-serif font, therefore it's easy to read. I have chosen this font beacause it looks good, and is easy to read. The backup fonts are Tahoma, Geneva, Verdana and sans-serif. I used these beacause my code editor recommended them.
